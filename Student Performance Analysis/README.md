# 📊 Student Performance Analyzer

## 📌 Project Overview
This project analyzes student academic performance using **Python, NumPy, Pandas, and Matplotlib**.  
The goal is to clean the data, calculate total and average scores, assign grades, identify top-performing students, analyze subject-wise performance, and visualize key insights.

---

## 📂 Dataset Description
The dataset contains marks of students in five subjects:
- Maths
- Science
- English
- Social
- Computer

Each row represents one student and their subject-wise scores.

---

## 🎯 Objectives
- Handle missing marks using subject-wise averages
- Perform row-wise calculations to compute:
  - Total score
  - Average score
- Assign grades using conditional logic
- Identify top 3 students based on total score
- Find subject-wise highest and lowest marks
- Create visualizations for performance insights
- Export cleaned and summarized data

---

## 🛠 Technologies Used
- Python
- NumPy
- Pandas
- Matplotlib

---

## 📚 Concepts Applied
- Data cleaning with `fillna()`
- Row-wise aggregation using `axis=1`
- Conditional logic using `np.where()`
- Data exporting using `to_csv()`
- Data visualization using Matplotlib

---

## ⚙️ Steps Performed
1. Loaded the dataset using Pandas
2. Checked and handled missing values
3. Calculated total and average scores
4. Assigned grades based on average marks
5. Identified top 3 students
6. Generated subject-wise highest and lowest marks
7. Created performance visualizations
8. Saved cleaned and summary datasets

---

## 📈 Visualizations
- Grade distribution chart
- Subject-wise average marks
- Total score distribution
- Top students comparison

---

## 📈 Output Files
- `cleanedStudentPerformance.csv` → Cleaned dataset
- `toppers.csv` → Top 3 students
- `subject_summary.csv` → Subject-wise min & max marks
- Visualization plots (PNG files)
